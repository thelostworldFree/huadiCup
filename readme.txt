Comc can be accessed via http://comc.herokuapp.com/

To run the application:
- Launch MongoDB with the mongod command
- Run the following commands to launch the application:
	- npm install
	- node server.js
- Access the application with localhost:3000

Please note: Admin features are not found by accessing localhost:3000/admin. Admins are able to exercise their privileges through accessing other users' profiles.
